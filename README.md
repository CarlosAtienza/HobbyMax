# HobbyMax

HobbyMax is a mobile application that helps users track their hobbies, log activity, and level up over time.  
It turns personal growth and consistency into a structured, game-like experience.

This repository contains the **frontend** built with React Native and Expo.

---

## Features

- User authentication (JWT)
- Create, update, and delete hobbies
- Log hobby sessions and progress
- XP and leveling system per hobby
- User profiles with hobby statistics
